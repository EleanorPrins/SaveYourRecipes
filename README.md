<h2>What's this about?</h2>
<p>
    SaveYour.Recipes is a self-hosted recipe manager, with a suite of other helpful
    accessories like a weekly meal planner and a shopping list which pulls from your
    recipes database! You can also easily add recipes from other sites by using the
    import tool or add one manually using the wizard.
</p>

<h2>Key Features</h2>
<ul>
    <li><strong>Recipe Management:</strong> <i>Create recipes via import or manually and manage them</i></li>
    <li><strong>Meal Planner:</strong> <i>Plan meals for the next week</i></li>
    <li><strong>Shopping List:</strong> <i>Shop for ingredients based on your recipes</i></li>
    <li><strong>Docker/Container Registry:</strong> <i>Easy deployment and updating with Docker and the Container Registry</i></li>
</ul>
